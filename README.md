# Heart Attack Risk Level Prediction

## Project Overview
This project predicts the risk level of heart attack using machine learning techniques. The goal is to analyze patient health data and classify individuals into different risk categories.

## Objectives
- Predict heart attack risk
- Compare machine learning models
- Evaluate model performance

## Tools and Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest

## Dataset
Heart Attack Risk Dataset

## Results
The model predicts heart attack risk based on patient health parameters with good accuracy.
